Onboarding

5 steps, 0 headaches

Setting up the Rescale AI bridge is incredibly simple. Just run the local executable and let the extension do the rest.

1

Download the repository

Download and extract the latest release files containing the local bridge and browser extension.

2

Load the extension

Go to your browser extensions page, toggle "Developer mode" on, and load the unpacked rescale-ai-extension folder.

3

Enable MCP in Roblox Studio

Open Studio, load a Place, go to Assistant AI settings and click Enable Studio as MCP Server.

4

Start the bridge

Double-click the start.bat script to run the local websocket bridge connecting your browser to Studio.

5

Open your AI chat

Go to ChatGPT, Claude, or DeepSeek. Click Start Roblox agent and start chatting to build your game!
